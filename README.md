Live Demo https://11ty-mix.netlify.app/

## Usage

```bash
npm install 
# or
yarn install
```

Then you can:

```bash
# Start dev and run website on http://localhost:8080
npm run dev
# or
yarn run dev

# Build production website inside `static/dist`
npm run build 
# or
yarn run build
```
